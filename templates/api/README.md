# Fluxor API Template

A complete RESTful API template with JWT authentication, CORS support, and rate limiting.

## Features

- ✅ JWT Authentication
- ✅ CORS Support
- ✅ Rate Limiting
- ✅ RESTful API Structure
- ✅ Error Handling
- ✅ Input Validation
- ✅ Pagination

## Installation

```bash
composer create-project lizzyman04/fluxor-php my-app
cd my-app
```

## Development

```bash
composer dev
```

Visit http://localhost:8000
